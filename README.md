**Краткое описание проекта**  
`login_and_logout_test.py` - тесты логина и логаута  
`place_an_add_test.py` - тесты создания объявления  
`registration_test.py` - тесты регистрации

`helpers.py` - статические методы для создания аккаунта, используются в некоторых тестах  
`conftest.py` - файл с фикстурами  
`data.py` - данные для тестов
